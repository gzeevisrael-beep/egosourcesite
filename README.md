# Ищем способ преодолеть барьер своего эгоизма с помощью Источника

Адаптивный одностраничный сайт на русском языке.

## Запуск

Откройте файл `index.html` в браузере.

Для локального сервера:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Состав

* `index.html` структура и содержание
* `styles.css` оформление и адаптивность
* `script.js` анимации, интерактивная практика и атмосферный звук

Сайт не требует API ключа и не отправляет данные на сервер. Записи в поле практики остаются только в текущей странице браузера.
